# 🎬 Tamil Movie Explorer

A modern, responsive Movie Search Web Application built using **HTML, CSS, and JavaScript**. This project showcases a collection of Tamil movies with search, filtering, sorting, favorites, and movie details in a clean Netflix-inspired interface.

---

## 📸 Preview

Tamil Movie Explorer provides a simple and elegant interface for browsing Tamil movies.

---

## ✨ Features

- 🔍 Live Movie Search
- 🎭 Filter Movies by Genre
- ⭐ Sort by IMDb Rating
- 📅 Sort by Release Year
- 🔤 Sort Alphabetically
- ❤️ Add/Remove Favorites
- 💾 Favorites saved using Local Storage
- 📄 Movie Details Popup
- 🌙 Netflix-inspired Dark Theme
- 📱 Fully Responsive Design
- ⚡ Fast and Lightweight
- 🖥️ Single HTML File (No Framework Required)

---

## 🛠️ Built With

- HTML5
- CSS3
- JavaScript (ES6)

## 🚀 Getting Started

1. Download the project.
2. Open **index.html** in any modern web browser.
3. Start searching and exploring Tamil movies.

No installation required.
